# Almhini – Retail Management Desktop Application

**Almhini** is a Java-based desktop application tailored for retail businesses (e.g., mobile phones, perfumes) to manage inventory and sales efficiently.

## Key Features

- Manage product categories and individual items
- Record sales transactions and generate invoices
- View and analyze sales statistics

## Technologies Used

- **Java** (Swing for the graphical user interface)
- **MySQL** (for persistent data storage)

## Setup Instructions

To run this application, please ensure the following:

- A MySQL server is installed and running.
- Create a database named **`store`** on your MySQL server before launching the application.
