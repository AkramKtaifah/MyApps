# Mittagstisch – Helper Scheduling Desktop App

**Mittagstisch** is a Java-based desktop application designed to facilitate the planning and management of monthly helper schedules for meal services.

## Key Features

- Manage detailed information about helpers
- Plan workdays for each individual helper
- Generate and edit monthly duty schedules
- Export schedules as PDF documents

## Technologies Used

- **Java** (Swing for the graphical user interface)
- **MySQL** (for persistent data storage)

## Purpose

The application is built to support organizations or teams that rely on a rotating group of helpers, making it easier to assign tasks, track availability, and produce a clear, printable schedule for each month.

---

**Mittagstisch** streamlines the scheduling process and reduces administrative effort by combining a user-friendly interface with flexible planning tools.
